Webserv en cours d’exécution

Bienvenue sur Webserv

Serveur HTTP/1.1 (projet 42) — sockets, non-bloquant, multiplexage, parsing de requêtes et réponses. Cette page est servie depuis srcs/http/Pages/index.html.

GET / GET /index.html the goat REDIRECTIONS Upload Delete

Commandes rapides

curl -i http://localhost:8080/
curl -i http://localhost:8080/index.html

Pour tester les méthodes supportées : GET, POST, DELETE. Le chemin côté serveur pointe actuellement sur le dossier srcs/http/Pages.

POST /upload (exemple)

curl -i -X POST http://localhost:8080/upload \
  -H 'Content-Type: text/plain' \
  --data 'hello webserv'

DELETE (exemple)

# Supprime un fichier dans Pages si présent
curl -i -X DELETE http://localhost:8080/monfichier.txt

Fonctionnalités

  • HTTP/1.1
  • poll()
  • Non-bloquant
  • GET / POST / DELETE
  • Pages d’erreur

Endpoints utiles

  • / → /index.html
  • /index.html
  • /upload

Si tu obtiens 404, vérifie que le fichier existe dans srcs/http/Pages.

Build & run

make
./webserv srcs/config/default.conf

Port par défaut: 8080.
